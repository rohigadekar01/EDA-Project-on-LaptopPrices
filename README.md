# EDA-Project-on-LaptopPrices
This project focuses on Exploratory Data Analysis (EDA) of a laptop price dataset. The goal is to uncover insights, identify patterns, and understand how different specifications (like RAM, Processor, HDD/SSD storage, GPU, and Brand) influence laptop prices.
